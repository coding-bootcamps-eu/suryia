# Suriya-Api

## Requirements

Node.js v18.17.1.

## Getting started

### Install

```sh
npm install
```

### Prepare .env

```sh
cp .env.example .env
```

#### Define App

The Api requires a port. Define it in the `.env` file Example:

```
PORT=8080
```

### Start Dev Server

```sh
npm run dev
```
